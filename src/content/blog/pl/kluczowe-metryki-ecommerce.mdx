## Dlaczego metryki są ważne?

> "Co nie jest mierzone, nie może być poprawione" - Peter Drucker

W e-commerce intuicja to za mało. Musisz **wiedzieć**, nie **zgadywać**:
- Które produkty są rentowne
- Skąd przychodzą najlepsi klienci
- Ile możesz wydać na pozyskanie klienta
- Czy Twój biznes się rozwija czy kurczy

## Metryki sprzedażowe

### 1. Conversion Rate (CR) - Wskaźnik konwersji

**Wzór:**
```
CR = (Liczba transakcji / Liczba odwiedzin) × 100%
```

**Benchmark:**
- E-commerce ogółem: 2-3%
- Mobile: 1-2%
- Desktop: 3-4%
- Top sklepy: 5-10%

**Jak poprawić:**
- Lepsze zdjęcia produktów
- Jasne CTA (Call To Action)
- Uproszczony checkout
- Social proof (opinie, certyfikaty)

---

### 2. Average Order Value (AOV) - Średnia wartość zamówienia

**Wzór:**
```
AOV = Całkowity przychód / Liczba zamówień
```

**Przykład:**
- Przychód: 100,000 PLN
- Zamówienia: 500
- AOV = 200 PLN

**Jak poprawić:**
- Cross-selling (\"klienci kupili również\")
- Bundling (zestawy produktów)
- Darmowa dostawa powyżej progu
- Upselling (lepsze wersje produktu)

---

### 3. Revenue Per Visitor (RPV)

**Wzór:**
```
RPV = Całkowity przychód / Liczba odwiedzin
```

**Dlaczego ważne:**
Łączy CR i AOV w jedną metrykę. Pokazuje ile zarabiasz na każdym odwiedzającym.

**Przykład:**
- 10,000 odwiedzin
- 100,000 PLN przychodu
- RPV = 10 PLN

Oznacza: każdy odwiedzający to średnio 10 PLN przychodu.

---

## Metryki marketingowe

### 4. Customer Acquisition Cost (CAC) - Koszt pozyskania klienta

**Wzór:**
```
CAC = Koszt marketingu / Liczba nowych klientów
```

**Przykład:**
- Budżet marketingowy: 10,000 PLN/m-c
- Nowi klienci: 200
- CAC = 50 PLN

**Benchmark:**
- Fashion: 30-60 PLN
- Elektronika: 80-150 PLN
- Luksusowe produkty: 200-500 PLN

**Zasada:**
```
CAC < 30% × LTV
```

---

### 5. Return on Ad Spend (ROAS)

**Wzór:**
```
ROAS = Przychód z reklam / Koszt reklam
```

**Przykład:**
- Wydałeś 1,000 PLN na Meta Ads
- Sprzedałeś za 5,000 PLN
- ROAS = 5x (czyli 500%)

**Benchmark:**
- Minimum: 2x (zwrot 200%)
- Dobry: 4x (400%)
- Świetny: 6x+ (600%+)

---

### 6. Click-Through Rate (CTR) - Klikalność

**Wzór:**
```
CTR = (Liczba kliknięć / Liczba wyświetleń) × 100%
```

**Benchmark Google Ads:**
- Search ads: 3-5%
- Display ads: 0.5-1%
- Shopping ads: 0.5-1%

**Benchmark Meta Ads:**
- Feed ads: 1-2%
- Stories: 0.5-1%

---

## Metryki klienta

### 7. Customer Lifetime Value (LTV)

**Wzór (uproszczony):**
```
LTV = AOV × Średnia liczba zakupów rocznie × Średni czas życia klienta (lata)
```

**Przykład:**
- AOV: 200 PLN
- Zakupy/rok: 3
- Życie klienta: 2 lata
- LTV = 200 × 3 × 2 = 1,200 PLN

**Dlaczego ważne:**
Wiedząc LTV, wiesz ile możesz wydać na pozyskanie klienta (CAC).

**Złota zasada:**
```
LTV / CAC > 3
```

---

### 8. Repeat Purchase Rate - Wskaźnik powtórnych zakupów

**Wzór:**
```
Repeat Rate = (Klienci którzy kupili >1 raz / Wszyscy klienci) × 100%
```

**Benchmark:**
- Nowe sklepy: 5-15%
- Dojrzałe sklepy: 20-40%
- Top tier: 50%+

**Jak poprawić:**
- Email marketing (especially automation)
- Program lojalnościowy
- Retargeting
- Doskonała obsługa klienta

---

### 9. Churn Rate - Wskaźnik odejścia

**Wzór:**
```
Churn = (Klienci utraceni / Klienci na początek okresu) × 100%
```

**Przykład:**
- Styczeń: 1,000 klientów
- Luty: 900 klientów (100 nie wróciło)
- Churn = 10%

**Cel:** Churn < 5% miesięcznie

---

## Metryki operacyjne

### 10. Inventory Turnover - Rotacja zapasów

**Wzór:**
```
Turnover = Koszt sprzedanych towarów / Średni stan magazynowy
```

**Przykład:**
- COGS rocznie: 500,000 PLN
- Średni magazyn: 100,000 PLN
- Turnover = 5x/rok

**Co to znaczy:** Zapasy odnawiają się 5 razy w roku (co 73 dni).

**Benchmark:**
- Fashion: 4-6x/rok
- Elektronika: 8-12x/rok
- Artykuły spożywcze: 20-30x/rok

---

### 11. Fulfillment Time - Czas realizacji

**Metryka:**
Średni czas od zamówienia do wysyłki.

**Benchmark:**
- Same-day: &lt;24h
- Standard: 24-48h
- Slow: &gt;48h

**Dlaczego ważne:**
Szybka dostawa = wyższa satysfakcja = lepsze recenzje = wyższa konwersja.

---

### 12. Cart Abandonment Rate - Porzucone koszyki

**Wzór:**
```
Abandonment Rate = (1 - Transakcje / Koszyki stworzone) × 100%
```

**Benchmark:**
Średnia: **70%** (tak, 70% koszyków jest porzucanych!)

**Główne przyczyny:**
- Nieoczekiwane koszty wysyłki (55%)
- Brak konta nie wymagany (34%)
- Zbyt długi proces checkout (26%)
- Brak zaufania (17%)

**Rozwiązanie:**
- Email recovery (odzyskuje 10-15% porzuconych)
- Jasne koszty od początku
- Guest checkout
- Trust badges

---

## Metryki finansowe

### 13. Gross Margin - Marża brutto

**Wzór:**
```
Gross Margin = ((Przychód - COGS) / Przychód) × 100%
```

**Przykład:**
- Sprzedaż: 1,000 PLN
- COGS: 600 PLN
- Margin = 40%

**Benchmark:**
- Produkty własne: 40-70%
- Dropshipping: 15-30%
- Wholesale: 20-40%

---

### 14. Net Profit Margin - Marża netto

**Wzór:**
```
Net Margin = (Zysk netto / Przychód) × 100%
```

**Co uwzględnia:**
- Przychód
- MINUS: COGS
- MINUS: Marketing
- MINUS: Fulfillment, shipping
- MINUS: Platform fees
- MINUS: Team, tools
- = Zysk netto

**Benchmark e-commerce:**
- Nowe sklepy: 0-5%
- Dojrzałe: 10-20%
- Top performers: 20-30%

---

### 15. Cash Flow - Przepływ gotówki

**Metryka:**
Czy masz więcej pieniędzy na koniec miesiąca niż na początku?

**Dlaczego krytyczne:**
Możesz mieć świetne wyniki na papierze ale ujemny cash flow (np. długie terminy płatności od klientów, krótkie dla dostawców).

**Monitoruj:**
- Cash in (przychody)
- Cash out (koszty)
- Net cash flow

**Cel:** Pozytywny cash flow każdego miesiąca.

## Jak zacząć śledzić metryki?

### Dashboard podstawowy (Must-have)

1. **Dziś:**
   - Przychód
   - Zamówienia
   - Conversion Rate

2. **Ten miesiąc:**
   - Przychód vs poprzedni miesiąc
   - ROAS
   - Top 5 produktów

3. **Trendy:**
   - Wykres sprzedaży (ostatnie 30 dni)
   - CAC trend
   - Repeat rate

### Narzędzia

**Darmowe:**
- Google Analytics (traffic, CR)
- Google Search Console (SEO)
- Platforma e-commerce (Shopify Analytics)

**Płatne:**
- LiveSales (unified dashboard)
- Google Data Studio (custom reports)
- Mixpanel (advanced analytics)

### Częstotliwość przeglądu

- **Codziennie:** Przychód, zamówienia, CR
- **Co tydzień:** ROAS, CAC, top produkty
- **Co miesiąc:** LTV, churn, margins

## Podsumowanie

Nie musisz śledzić wszystkich 15 metryk od razu. Zacznij od **Top 5:**

1. **Revenue** (przychód)
2. **Conversion Rate** (konwersja)
3. **CAC** (koszt klienta)
4. **ROAS** (zwrot z reklam)
5. **Gross Margin** (marża)

Te 5 dadzą Ci 80% potrzebnego obrazu biznesu.

W miarę rozwoju dodawaj kolejne: LTV, repeat rate, inventory turnover.
